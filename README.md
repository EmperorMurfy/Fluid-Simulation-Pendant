## Overview
Since January, I've been particularly inspired by Mitxela's Fluid Simulation Pendant.
A watch-sized device, it contains it's own charlieplexed LED display driven by an STM32.
This allows the device to run a particle simulation that responds to gravity in real time.
In other words, water is displayed on it's LED display that flows and settles naturally as you move.
 
As a beginner, I was thoroughly intrigued. This came at the perfect opportunity as I sought to enter electronic design.
So, with no former experience, I set out to build my own from scratch.

You can find my full devlog at [Blueprint](https://blueprint.hackclub.com/projects/14158)
--

## PCB
This project was designed in KiCAD. V1 utilized a lilypad design with two 2-layer daughter boards (see devlog).
V2 features a singular 4-layer board, merging V1's two daughter boards.

## Setup

OBTAIN PCB
1) Download project files from Github. Use KiCAD to open hardware files.
2) Export & Upload to JLCPCB through Fabrication Toolkit
3) Select PCBA. Make sure you click on Standard + assemble on both sides. 
4) Follow on screen instructions. Verify BOM contents.
<img width="1684" height="674" alt="Screenshot 2026-03-31 233642" src="https://github.com/user-attachments/assets/e201175d-a503-4396-959d-5dd9e4f1531d" />
5) Purchase and await arrival.

While waiting, you can 3D print the case.

ASSEMBLY 
1) Solder PH1 pins on the back of the PCB at drilled holes labeled BOOT0, NRST, DEBUG_TMS, GND, DEBUG_TCK, 3.3V
2) Connect to Nucleo's ST-LINK, Connect to PCB
3) Install STM32CubeProgrammer and STM32CubeIDE
4) Open software folder from downloaded project files, select STM32.ioc (this should open STM32CubeProgrammer)
5) Select generate code (this should open STM32CubeIDE)
6) Replace the generated main.cp or cpp with main.cp from project files, click run to save
8) Install PCB to case - Be CAREFUL! Putting it at an angle with the USB-C pointed towards the opening helps.
9) Plug the battery into PCB's JST-2 port
10) Install the back lid + push in the watch glass
