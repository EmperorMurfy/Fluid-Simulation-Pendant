# Fluid-Simulation-Pendant
A Fluid Simulation Pendant Inspired by Mitxela

## Introduction
Since January, I've been particularly inspired by Mitxela's Fluid Simulation Pendant.
A watch-sized device, it contains it's own charlieplexed LED display driven by an STM32.
This allows the device to run a particle simulation that responds to gravity in real time.
In other words, water is displayed on it's LED display that flows and settles naturally as you move.
 
As a beginner, I was thoroughly intrigued. This came at the perfect opportunity as I sought to enter electronic design.
So, with no former experience, I set out to build my own from scratch.

You can find my full devlog at [Blueprint](https://blueprint.hackclub.com/projects/14158)
